# LargeScreenCalendar
## 预览图
![预览图](https://github.com/YuxiangZhu/LargeScreenCalendar/blob/master/images/preview.jpg "日历预览")
## 背景
* 安卓大屏机32寸、1080*1920分辨率、默认读取mdpi
- 售票系统，对日期的要求是最早只能买今天，最迟的可配置
* 固定分辨率，大量使用px和设计稿保持一致，否则锯齿严重
- 纯净版局限性很大，不适用于手机端
